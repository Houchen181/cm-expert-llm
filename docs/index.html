🧊 CondensAI

Domain-expert LLM toolkit for condensed matter physics. Build, train, and deploy physics-specialized language models with RAG and LoRA fine-tuning.


View on GitHub Quick Start

Why CondensAI?

🎯 Domain Specialized

Trained on real condensed matter physics papers, textbooks, and research. Outperforms general LLMs on physics-specific questions.

📚 RAG-Enhanced

Retrieval-augmented generation from actual physics literature with citations to original papers.

🚀 Easy Deployment

One-click Colab demo, Streamlit UI, Docker deployment, and Hugging Face Spaces integration.

📊 Evaluated

CMPhysBench framework with 400+ questions across undergraduate, graduate, and research levels.

💻 Consumer GPU Friendly

LoRA fine-tuning with 4-bit quantization. Train on a single consumer GPU.

🔬 Research Ready

Complete with unit tests, CI/CD, documentation, and academic citation support.

Quick Start (5 minutes)

# Step 1: Install
pip install cm-expert-llm

# Step 2: Verify installation
python -m cm_expert.verify

# Step 3: Run demo
python examples/01_basic_usage.py

# Step 4: Ask physics questions
python -m cm_expert ask "What is the Meissner effect?"
