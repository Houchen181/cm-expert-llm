# Contributor Spotlight

This page celebrates the amazing people who contribute to CondensAI!

## Why Contributors Matter

CondensAI exists because of contributors like you. Every contribution—whether it's fixing a typo, adding a test, writing documentation, or submitting physics content—makes this project better and more useful for everyone.

## How to Get Featured

You'll be featured here when you:
- Submit your first PR (any size!)
- Report a critical bug
- Improve documentation significantly
- Add physics content (papers, notes, problems)
- Help others in Discussions or Issues

## How to Contribute

### Easy Ways to Start

1. **Fix a typo** in documentation
2. **Add a test** for existing code
3. **Improve an error message**
4. **Add physics examples** from your research
5. **Answer a question** in Discussions

## Recognition Levels

We recognize contributions at different levels:

| Level | Requirement | Recognition |
|-------|-------------|-------------|
| 🥉 First PR | Submit your first merged PR | Name in this file |
| 🥈 Contributor | 3+ merged PRs | Name + contribution area |
| 🥇 Core Contributor | 10+ merged PRs or major feature | Permanent section, reviewer status |
| ⭐ Maintainer | Significant ongoing contributions | Commit access, project decisions |

## Benefits of Contributing

- **Visibility**: Your work is seen by physicists worldwide
- **Learning**: Improve your ML and physics skills
- **Networking**: Connect with other researchers
- **Career**: Open-source contributions look great on CVs
- **Impact**: Help democratize AI for physics research
